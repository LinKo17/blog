There are two parts in this project.(user part and admin part)
There are two types of user in this blog website.
If users aren't registered, they can read the blog but they can't create their own blog and
also can't write comment , they can't do a lot of other things either.
If users are registered, they can read the blog ,create their own blog , write comments ,
they can do a lot of other things either.
In the admin section, Admin has his own dashboard . So he can manage users.
Admins can approve user's post , check the post , delete the post , ban the users ,send to
user from the website and also can do a lot of other things either.
I built authentication system ,authorization system , email validation system and sending
email from website system for this website .
